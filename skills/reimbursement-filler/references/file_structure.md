# 文件结构要求

为了确保 `reimbursement-filler` 技能正常运行，工作区应保持以下结构：

## 输入文件

1. **火车票提取结果_最新.xlsx**
   - 必须包含 `price` 列（第12列）。
   - 金额可以是数值，也可以是形如 `=720.00` 的字符串公式。

2. **滴滴电子发票汇总.xlsx**
   - 必须包含 `金额` 列。

3. **费用报销单模板.xlsx**
   - 预设的 Excel 报销单模板。
   - `E5`: 交通费填入位置。
   - `D3`: 日期填入位置。
   - `J3`: 单据页数填入位置。

## 附件文件夹

1. **滴滴出行电子发票及行程报销单/**
   - 存放所有滴滴相关的 PDF 附件。

2. **火车票/**
   - 存放所有火车票相关的 PDF 附件。
